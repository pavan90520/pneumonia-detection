# Pneumonia Detection

Deep learning model for pneumonia detection using transfer learning (DenseNet121) and Grad-CAM.  
The notebook includes preprocessing, model training, evaluation, and explainability.  
This project is part of my AI/ML engineering portfolio.

---

## Dataset  
This project uses the publicly available Chest X-Ray Pneumonia dataset from Kaggle:  
https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia

---

## Key Features  
- DenseNet121 transfer learning  
- Data augmentation and early stopping  
- Achieved ~98.22% test accuracy  
- Grad-CAM heatmaps for model interpretability  
- Clear, reproducible training workflow in Jupyter Notebook
